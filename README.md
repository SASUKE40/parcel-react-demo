# Parcel React Demo

## Guild

```bash
git clone git@github.com:SASUKE40/parcel-react-demo.git
cd parcel-react-demo
npm i
npm start
```

## Build

```bash
npm run build
```

## Serve application

Before serving, you should build the application firstly.

```bash
npm run serve
```

open [http://localhost:5000/](http://localhost:5000/) to see.
